## 사용기술

- **언어** :  JAVA, PHP

- **웹서버** :  Apache, Nginx

- **서버 호스팅** :  AWS

- **데이터베이스** :  Mysql

- **프로토콜** :  http, tcp/ip, rtmp

- **라이브러리 및 API** :  

  - 통신

    Retrofit2

  - 이미지

    Glide, circleimageview

  - 동영상

    Exoplayer

  - 소셜 로그인 

    Facebook Login API, Kakao Login API

  - 웹 크롤링, 결제

    Jsoup, 아임포트
  
  - 권한

    TedPermission


- **참고한 오픈소스**

  - 영상통화 : https://github.com/webrtc/apprtc
  - 방송하기 : https://github.com/TakuSemba/docker-nginx-rtmp
