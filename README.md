# Streaming

사용자들 간 동영상을 공유하고, 방송을 통하여 소통 할 수 있는 어플

## 사용기술



- **언어** :  JAVA, PHP

- **웹서버** :  Apache, Nginx

- **서버 호스팅** :  AWS

- **데이터베이스** :  Mysql

- **라이브러리 및 API** :  

  - 통신

    Retrofit2

  - 이미지

    Glide, circleimageview

  - 동영상

    Exoplayer

  - 소셜 로그인 

    Facebook Login API, Kakao Login API

  - 웹 크롤링, 결제

    Jsoup, 아임포트
  
  - 권한

    TedPermission

- **참고한 오픈소스**

  - 영상통화 : https://github.com/webrtc/apprtc
  - 방송하기 : https://github.com/TakuSemba/docker-nginx-rtmp
